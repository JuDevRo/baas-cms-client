Instrucciones frontend

Seguimos trabajando en el componente de Projects.

Ahora tenemos que traer el schema desde el backend con esta ruta /scheme/:userId/:projectId y en el response.data.structure se guarda la estructura de los campos del schema.

1- Traer el schema desde la petición del backend con la ruta /scheme/:userId/:projectId.

2- Mostrar los campos de la structure, los campos tienen que tener el siguiente formato si es que existen, si esta vacio mostrar un mensaje de que no hay campos

response.data.structure = [
  {
    key: "title",
    label: "Título",
    type: "text",
    required: true,
    multiple: false
  },
  {
    key: "galeria",
    label: "Galería",
    type: "image",
    multiple: true
  },
  {
    key: "precio",
    label: "Precio",
    type: "number",
    required: true
  },
  {
    key: "lanzamiento",
    label: "Fecha de lanzamiento",
    type: "date"
  },
  {
    key: "portada",
    label: "Portada",
    type: "image",
    multiple: false
  }
]

3- Crear el modal puedes ver ejemplo de como usarlo en el componente Home, al clickar en add fields se abre un modal para poder agregar campos y modificar esa structure del scheme, haz la logica de modificación y guarda los cambios en la siguiente ruta /scheme/update/:userId/:id metodo patch

Usa el index.css para entender la logica de estilos.

Todo tiene que ser responsive
