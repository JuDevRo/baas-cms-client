Instrucciones de frontend

Añade a las tarjetas de proyecto un boton con ... en la esquina superior derecha, al clickarlo se abre un popover con la opcion editar o eliminar.
Si das a eliminar abre un modal preguntando "Are you sure you want to delete Blog Posts? This action cannot be undone and all entries will be permanently deleted.", junto con la opcion cancel y delete.

Y luego en el boton edit se puede cambiar el name o la descripcion no hay mas que editar.

Busca el contexto necesario para realizar bien la tarea
